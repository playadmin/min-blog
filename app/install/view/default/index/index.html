min-Blog 安装程序

环境要求：

  1. PHP：7.1 以上
  2. Mysql：5.7 以上 或 MariaDB 10.1 以上
  3. Apache2.4 以上 或 Nginx 1.15 以上
  4. 需要伪静态支持
Apache 伪静态配置：

<IfModule mod_rewrite.c>

    RewriteEngine on

    RewriteRule ^ad/(.*)|ad$ ad/index.html [L]

    RewriteRule ^(.+\.[\w-\d]+)(.*)$ $1 [L]

    RewriteRule ^(.+)$ index.php [L,E=PATH_INFO:$1]

</IfModule>

Nginx 伪静态配置：

location ~* ^/((?!\.[\w-\d]).)*$ {

    rewrite ^/ad/(.*)|/ad$ /ad/index.html last;

    rewrite ^/(.+)$ /index.php/$1 last;

}
